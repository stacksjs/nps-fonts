# @nps-fonts/nps-symbols

NPS Symbols — Original parametric pictograph icon font — 23 NPS-themed symbols.

Original parametric font, drawn from scratch by NPS Fonts contributors. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/nps-symbols
# or: npm install @nps-fonts/nps-symbols
```

## Use

```css
@import "@nps-fonts/nps-symbols";

body { font-family: "NPS Symbols", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/nps-symbols/index.css">
```
