# @nps-fonts/sequoia-sans

Sequoia Sans — High-contrast humanist sans for park field guides — variable weight Thin → Wide.

Original parametric font, drawn from scratch by NPS Fonts contributors. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/sequoia-sans
# or: npm install @nps-fonts/sequoia-sans
```

## Use

```css
@import "@nps-fonts/sequoia-sans";

body { font-family: "Sequoia Sans", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/sequoia-sans/index.css">
```
