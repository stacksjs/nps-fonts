# @nps-fonts/campmate-script

Campmate Script — Soft rounded upright script with ligatures — perfectly imperfect trailhead lettering.

Original parametric font, drawn from scratch by NPS Fonts contributors. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/campmate-script
# or: npm install @nps-fonts/campmate-script
```

## Use

```css
@import "@nps-fonts/campmate-script";

body { font-family: "Campmate Script", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/campmate-script/index.css">
```
